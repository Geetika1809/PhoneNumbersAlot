# README


Command-

1. bundle install
2. rake db:migrate
3. rake db:seed
4. {BASE_URL}/api/v1/phones # get all alloted phones
5. {BASE_URL}/api/v1/phones # post- allot phones to client(not dynamic client id is 1)
6. {BASE_URL}/api/v1/phones/:phone_number # post- allot phone_number to client
